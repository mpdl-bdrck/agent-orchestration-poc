## Description

Create the JSON file containing mock alert templates that simulate real API responses. This file will drive the notification panel behavior and support multiple playback modes.

## Acceptance Criteria

- [ ] File `config/notifications/mock_alerts.json` exists
- [ ] JSON structure includes:
  - [ ] `metadata` section (source, version, description)
  - [ ] `alerts` array with at least 4 alert templates
  - [ ] `settings` section (mode, loop, interval_seconds, max_alerts)
- [ ] Alert templates include all required fields:
  - [ ] `id`, `agent`, `issue_type`, `severity`, `message`
  - [ ] `campaign_id` or `deal_id` (optional)
  - [ ] `details`, `timestamp`, `delay_seconds`
- [ ] JSON validates successfully
- [ ] Includes examples for all 4 agents (guardian, optimizer, specialist, pathfinder)
- [ ] Includes examples for different severity levels (critical, warning, info)
