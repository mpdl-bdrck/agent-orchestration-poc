## Description

Implement support for all three playback modes (streaming, static, replay) and robust error handling in background monitor.

## Acceptance Criteria

- [ ] Streaming mode: gets next alert, sends message, waits interval
- [ ] Static mode: loads all alerts, sends all at once, waits 5 minutes
- [ ] Replay mode: uses delay_seconds from each alert for precise timing
- [ ] Loop support: resets loader and continues when alerts exhausted
- [ ] Error handling: catches exceptions, logs errors, waits 60s before retry
- [ ] Continues running even if individual alerts fail
- [ ] Logs mode and interval on startup
- [ ] Handles missing loader gracefully
