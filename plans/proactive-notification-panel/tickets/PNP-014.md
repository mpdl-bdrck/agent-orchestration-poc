## Description

Implement the click handler that sends context-injected messages to the Orchestrator when users click alerts.

## Acceptance Criteria

- [ ] Click handler exists on alert elements
- [ ] Generates `SYSTEM_TRIGGER:` message format:
  ```
  SYSTEM_TRIGGER: Focus on {entity_id}. Context: {issue_type}. {details}
  ```
- [ ] Uses `campaign_id` or `deal_id` as entity_id
- [ ] Includes `issue_type` from alert
- [ ] Includes `details` from alert
- [ ] Sends message via Chainlit API (`window.chainlit.sendMessage` or CustomEvent)
- [ ] Handles API errors gracefully
