## Description

Add robust JSON validation and error handling to the `NotificationLoader` class to ensure graceful degradation when JSON is missing or malformed.

## Acceptance Criteria

- [ ] Validates JSON structure on load
- [ ] Checks for required fields in metadata, alerts, and settings
- [ ] Validates alert structure (required fields present)
- [ ] Auto-creates default JSON if file missing
- [ ] Logs warnings for missing JSON (creates default)
- [ ] Raises `ValueError` if JSON structure invalid
- [ ] Logs errors but doesn't crash on load failures
- [ ] Provides helpful error messages for debugging
